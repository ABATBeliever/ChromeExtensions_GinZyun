## 銀盾Chrome拡張 GinZyun for EdgeAddon

銀盾は、きわめて単純なUIを持った検閲拡張機能です。

URL内にルールにあう文字列があった場合、接続を中止します。

・一時的無効化もできる

・広告なし、オープンソース

・*.xyz* のような表記に対応
